Japan Trip - Nara 奈良

日本千年古都 - 曾是平城時代日本首都，日本佛教發源地，境內佛寺眾多，擁有日本最早的古寺。
現乃為日本著名古都及國際觀光都市，是日本歷史與文化的搖籃之地。(鹿很多也不怕人的城市)

不用攜帶大行李、以輕便的小背包搭配球鞋即可

預計花費(不含吃飯) : ¥500(一日券) + ¥500(博物館平常展) + ¥150 * N (鹿仙貝 - 一群鹿會等著你餵食、或用搶的) + ( 選擇: 寺廟部分設施進入參觀是要收費 門票 ¥300~600 )
= ¥1000 + ¥150*N + ¥500*N

一日通行券 ¥500
(NTD : JPD = 1 : 3)

搭乘地鐵至近鐵奈良站(離古蹟及奈良公園較近)，JR 奈良站也可， 可能要轉巴士，或者步行10~20分鐘，參觀各地古蹟。(1~2公里)

一日券當日有效，上車時拿給司機看一下， 因為奈良是國際觀光都市，司機很和善， 如果不知道要從前門還是後門上車，就看司機停車時，你前面是前門還是後門。

淺綠色背景是著名觀光景點，不乏都是日本著名世界文化遺產。 深綠色線條是屬於巴士路線，圓圈是巴士有停的站。
